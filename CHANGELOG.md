# Changelog

本项目的所有重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## [0.6.0] - 2026-07-11

### 新增
- 组织归属校验：所有预约写操作（创建/更新/取消/审核）需验证用户归属对应组织
- `_check_org_admin` 和 `_check_org_member` 函数，校验组织级权限
- `_get_room_org_id` 辅助函数，通过房间 ID 查询组织 ID

## [0.5.0] - 2026-07-11

### 新增
- 校园全景 3D 场景（Campus.vue）
- 跨尺度缩放：校园→建筑详情页平滑过渡
- 面包屑导航：校园全景 / 校区名 / 建筑名
- 跨级搜索：首页搜索栏支持层级路径显示和点击导航
- 各级容器新建功能：管理员可创建建筑/楼层/房间，支持 3D 位置选择

## [0.4.0] - 2026-07-11

### 新增
- 空间利用率统计（柱状图、饼图、数据表格）
- 预约趋势分析（7/14/30 天折线图）
- 智能预约推荐（可用房间 + 最佳时段）
- 异常占用检测（过期未审核、频繁预约、潜在爽约）
- Stats.vue 数据统计页面

## [0.3.0] - 2026-07-11

### 新增
- BMTS Schema v0.1 JSON Schema 定义
- 数据导入功能（JSON/YAML，支持 UTF-8 BOM 和 GBK 编码）
- 数据导出功能（JSON/YAML）
- Schema 验证工具
- SchemaDoc.vue 规范文档页面

## [0.2.0] - 2026-07-11

### 新增
- 用户认证系统（注册/登录/JWT）
- 房间预约功能（冲突检测、过去时间校验）
- 预约审核流程（管理员通过/拒绝）
- 我的预约管理（查看/取消）
- 房间日程查询（30 分钟粒度）
- 预约对话框和时间轴可视化组件
- 三级权限体系（管理员/组织用户/访客）

### 修复
- 时区比较 TypeError：前端 naive datetime 与后端 aware datetime 不兼容
- 预约时间 CST→UTC 转换：前端时间存入 PostgreSQL 时 8 小时偏移

## [0.1.0] - 2026-07-11

### 新增
- 分级容器数据模型（校园→建筑→楼层→房间）
- 容器 CRUD API
- 3D 场景渲染（Three.js）
- 3D 交互选择（点击高亮 + 信息面板）
- 楼层切换功能
- 左侧导航树
- 校园信息卡片编辑
- 导航-3D 联动交互
